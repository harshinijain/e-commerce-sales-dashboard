# AI-Driven E-Commerce Sales Performance Analysis and Sales Prediction

AI-Driven E-Commerce Sales Performance Analysis and Sales Prediction using Python, Power BI, and Machine Learning. This project focuses on cleaning and analyzing e-commerce sales data, building interactive business intelligence dashboards, and developing a Linear Regression model to predict future sales. It provides insights into sales performance, profit, regional trends, category-wise analysis, and supports data-driven business decision-making.
